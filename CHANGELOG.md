# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-06

### Added
- Initial release with 8 core developer tools
- JSON formatter, minifier, escaper, unescaper
- Base64 encoder/decoder with file support
- URL encoder/decoder with parameter parser
- JWT decoder with expiration check
- Regex tester with common patterns library
- HTML entity encoder/decoder with reference table
- Hash calculator (MD5, SHA-1, SHA-256, SHA-512) with file support
- Timestamp converter with auto-detection
- Dark/light theme with system preference detection
- PWA support with offline capability
- Keyboard shortcuts (Ctrl+K search, Esc back)
- Responsive design for desktop and mobile
- GitHub Actions deployment workflow

### Security
- 100% offline processing
- Content Security Policy headers
- Security response headers (X-Frame-Options, X-XSS-Protection)
- No data collection or tracking
